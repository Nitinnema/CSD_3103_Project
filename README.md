# Bookstore NodeJS Application

Welcome to the Bookstore NodeJS Application - a comprehensive platform for book enthusiasts to explore, search, and purchase their favorite books. This application allows customers to sign up and log in, browse through a wide range of books, add them to their carts, place orders, and track them post-purchase. Additionally, admin users have access to exclusive functionalities such as Book Management, Inventory Management, and Order Management.

## Features

- **User Authentication**: Sign up and login functionality for customers.
- **Book Browsing and Search**: Explore and search for a wide variety of books.
- **Shopping Cart**: Add books to your personal cart.
- **Order Placement and Tracking**: Place orders and track their progress.
- **Admin Exclusive Features**: Special functionalities for admin users, including Book Management, Inventory Management, and Order Management.

## Installation

To get started with this application, follow these simple steps:

1. **Clone the Repository**:
   ```
   git clone https://github.com/OmarAbouzeid95/CSD_3103_Project.git
   ```

2. **Install Dependencies**:
   Navigate to the project directory and run:
   ```
   npm install
   ```

3. **Start the Application**:
   ```
   npm start
   ```
   Or alternatively:
   ```
   node index.js
   ```

4. **Access the Application**:
   Open your web browser and visit `localhost:3000` to interact with the application.

## Usage

After launching the application, you can:

- Create a new customer account or log in with existing credentials.
- Browse through the collection of books.
- Use the search feature to find specific books.
- Add books to your cart and proceed to checkout.
- As an admin, manage the book inventory and orders.
